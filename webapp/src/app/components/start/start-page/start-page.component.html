Välkommen

I behov av nya grupper till NO-projektet? Behövs lag till knypplingssällskapets årliga tipsrunda? Erfordras en rockad inom swingersföreningens konstellationer? SwiftGrouping tillgodoser dina behov av randomiserade grupper.

För att använda tjänsten behöver du en enhet. Välj mellan att skapa ny, ladda befintlig eller använd demoenheten för att testa applikaitonen.
